## Gulp 與 Webpack 開發工具
gulp+webpack+scss+babel

### 如何使用

***

Node 版本號 ： v8.9.4

npm 版本號 ： v5.6.0

***

安裝：

```
$ npm install
```

啟用：

```
$ npm run start
```

### 目錄結構

```
|-- src
   |-- scss
   |-- app
   |-- html
   |-- images
|-- dest
   |-- css
   |-- js
   |-- html
   |-- images
|-- .eslintrc          ## ESLint 相關設定
|-- gulpfile.js
|-- webpack.config.js
|-- package.json
|-- README.md
```
