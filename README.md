## Gulp 與 Webpack 開發工具


### 如何使用

***

Node 版本號 ： v5.7.0

npm 版本號 ： v3.6

***

安裝：

```
$ npm install
```

啟用：

```
$ npm run start
```

### 目錄結構

```
|-- src
   |-- sass
   |-- app
   |-- html
   |-- images
|-- dest
   |-- css
   |-- js
   |-- html
   |-- images
|-- app.js             ## express 路徑 router 設定
|-- .eslintrc          ## ESLint 相關設定
|-- gulpfile.js
|-- webpack.config.js
|-- package.json
|-- README.md
```
